### HTTP method
GET

### API endpoint
`/api/v1/chats/:chat_access_token/chat_memberships/chat_users`

### Request body
```
{
  "page": "1"                                                                   (integer, required)
}
```

### Response
#### Success
```
HTTP code 200 :ok
{
  "users: "[UserProfile]"                                                  (array of objects, required)
}
```
