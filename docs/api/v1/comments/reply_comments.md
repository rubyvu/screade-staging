### HTTP method
GET

### API endpoint
`/api/v1/comments/:id/reply_comments`

### Request body
```
{
  "page": 1                                                                     (integer, required)
}
```

### Response
#### Success
```
HTTP code 200 :ok
{
  "comments: [Comment]                                                          (array of objects, required)
}
```

#### Errors
- when Comment doesn't exists
```
HTTP code 404 :not_found
{
  "errors": ['Record not found.']                                               (array of strings, required)
}
```
