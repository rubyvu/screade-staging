### HTTP method
GET

### API endpoint
`/api/v1/user_assets/:username/images`

### Request body
```
{
  "page": "1"                                                                   (integer, required)
}
```

### Response
#### Success
```
HTTP code 200 :ok
{
  "images: "[UserImage]"                                                        (array of objects, required)
}
```

#### Errors
- when User doesn't exists
```
HTTP code 404 :not_found
{
  "errors": ['Record not found.']                                               (array of strings, required)
}
```
