### HTTP method
GET

### API endpoint
`/api/v1/user_assets/:username/videos`

### Request body
```
{}
```

### Response
#### Success
```
HTTP code 200 :ok
{
  "videos: "[UserVideo]"                                                        (array of objects, required)
}
```

#### Errors
- when User doesn't exists
```
HTTP code 404 :not_found
{
  "errors": ['Record not found.']                                               (array of strings, required)
}
```
