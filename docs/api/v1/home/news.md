### HTTP method
GET

### API endpoint
`/api/v1/home/news`

### Request body
```
{
  "is_national": true                                                           (boolen, optional)
  "page": 1                                                                     (integer, required)
  "location_code": "UA"                                                         (string, optional)
}
```

### Response
#### Success
```
HTTP code 200 :ok
{
  "news: "[NewsArticle]"                                                        (array of objects, required)
}
```
