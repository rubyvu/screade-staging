### Object structure
```
{
  "id" : "1"                                                                    (integer, required)
  "is_viewed": "false"                                                          (boolean, required)
  "message": "Notification message"                                             (string, required)
  "source": Comment || Event || Post || UserImage || UserVideo || SquadRequest  (object, required)
  "user": User                                                                  (object, required)
}
```
