### Object structure
```
{
  "id": "1"                                                                     (integer, required)
  "message": "Comment text example"                                             (string, required)
  "created_at": "2021-02-04 14:22:14 +0000",                                    (string, required, represents date in format "%Y-%m-%d %H:%M:%S %z")
  "user": "User"                                                                (object, required)
}
```
